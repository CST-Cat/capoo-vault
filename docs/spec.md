# 标注规范

## 概述

每张 Capoo GIF 贴纸由 AI 视觉模型标注，生成 7 个语义字段。

## 字段定义

### gif
GIF 文件名，格式为 `{序号}-{文件名}.gif`

### set
所属贴纸包名称，与 Telegram sticker set 对应

### emotion
主要情绪，1-2个中文词。常用情绪词：
- 正面：开心、兴奋、得意、撒娇、害羞、甜蜜、满足、温馨
- 负面：悲伤、愤怒、委屈、崩溃、惊恐、无语、疲惫、烦躁
- 中性：困惑、慵懒、好奇、呆萌、淡定

### action
动作描述，2-6个中文词。描述角色在做什么：
- 挥手打招呼、捂脸害羞、举爪欢呼、趴地大哭
- 抱着食物、跳舞、打瞌睡、吐舌头

### scene
场景描述，2-4个中文词。描述环境/背景：
- 白色背景、深色背景、餐桌前、浴室
- 纯色背景、室内、户外

### description
15-25个中文字符的一句话描述。要求：
- 完整描述贴纸内容
- 包含角色、动作、情绪、场景等关键信息
- 适合用于聊天匹配和搜索

### tags
5-8个中文标签。要求：
- 涵盖情绪、动作、角色、场景等维度
- 使用常用词汇，便于搜索匹配
- 避免过于专业的术语

## 质量标准

- 所有字段使用简体中文
- JSON 使用标准双引号
- 值内容中的引号使用「」
- tags 为数组格式，5-8个元素
- description 长度 15-25 字符
- 空返回（子任务未写入文件）会自动重试
